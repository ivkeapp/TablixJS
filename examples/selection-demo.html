TablixJS Selection Demo

Interactive demonstration of row selection functionality

How to Use Selection:

  • Single Mode:
    • Click any row to select it (previous selection will be cleared)
    • Click the selected row again to deselect it
  • Multi Mode:
    • Click a row to select it (clears other selections)
    • Click the only selected row to deselect it
    • Ctrl+Click to add/remove individual rows
    • Shift+Click to select a range from the last selected row
